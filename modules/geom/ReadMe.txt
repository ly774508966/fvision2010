geom
====

:author ferryzhou@gmail.com
:version 0.1.0

Overview
--------

geom provides a set of basic geometric utils and entities:

- geometric entities:
  - Line
  - PointPair
  - Camera // internal, external, full; 2D, 1D; parameter computation utils
- transformation of points/lines  // 2D->2D
- projection of points  //3D->2D, 2D->1D
- lens distortion
- rotation matrices  // rotation around x, y, z axes; creation and computation
- single view geometry utils
  - camera matrix
- two view geometry utils related
  - fundamental matrix
  - essential matrix
  - homography
  - triangulation
- simulation
