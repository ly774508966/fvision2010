cvutils
=======

:author ferryzhou@gmail.com
:version 0.1.0

Overview
--------

1. wrapped some opencv functions for easier use in c++, for instance, overloaded arithmic operators and stream operators for points and mats.
2. some frequently used geometry and matrix functions which are not in opencv.

contents includes:
- matrix utils. the output is usually in return values.
- stream operators for mats and points
- point arithmetic operators. 
- points/lines geometry functions.
- transformation between point and mat
- predefinded colors. for instance, Colors::RED
